# Javascript Questions

# Scope

```jsx
var var1 = 1;

if (true) {
  var var2 = 2;
  if (true) {
    var var3 = 3;
  }
}

function foo() {
  var var4 = 4;

  function bar() {
    var var5 = 5;
  }
}

console.log(var1);
console.log(var2);
console.log(var3);
console.log(var4);
console.log(var5);
```

<details>
  <summary>콘솔에 어떻게 찍힐까요?</summary>
  1<br />2<br />3<br />
  var4 is not defined
  <br />
  var5 is not defined
</details>
